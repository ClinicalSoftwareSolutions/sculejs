Class: CachingLinkedList

CachingLinkedList

{CachingLinkedList}

new CachingLinkedList(threshold, cacheKey, list) → {Void}

A proxy pattern implementation wrapping a linked list, this class provides an LRU cache for various access methods on the list - most notably search. It becomes an O(1) operation rather than O(n) once the cache is warmed up as long as the entry remains in the cache. See the LinkedList class for function documentation.
Parameters:
Name Type Description
threshold Number the LRU cache threshold - maximum number of entries
cacheKey String the object key to cache on
list LinkedList | DoublyLinkedList (optional) the linked list to encapsulate
